# 🏗️ 삼우에프엔지 교육센터 LMS

원본 Figma Make 디자인(https://amber-base-77385202.figma.site/) 기반으로 구현한  
기업 안전 교육 LMS 플랫폼입니다.

---

## 📁 파일 구조
```
index.html          ← 로그인 페이지
dashboard.html      ← 대시보드 (로그인 후)
css/login.css       ← 로그인 스타일
css/dashboard.css   ← 대시보드 스타일
js/login.js         ← 로그인 처리
js/data.js          ← 공통 데이터
js/dashboard.js     ← 대시보드 앱 로직
manifest.json / sw.js ← PWA 설정
```

---

## ✅ 구현 화면

### 🔐 로그인 (index.html)
- 물류창고 항공사진 배경 + 글래스모피즘 카드
- 삼우에프엔지 교육센터 로고 (방패+SW)
- 아이디/비밀번호 입력, 비밀번호 보기 토글
- 로그인 상태 유지 체크박스
- 회원가입 | 아이디찾기 | 비밀번호찾기 링크
- 푸터: COPYRIGHT © SAMWOO F&G ALL RIGHTS RESERVED

### 📊 대시보드 (dashboard.html) — 원본과 동일한 레이아웃
**상단 헤더**: 방패 로고 + "삼우에프엔지 교육센터" + 알림/로그아웃

**상단 탭 9개 (원본 동일)**:
| 탭 | 아이콘 | 특이사항 |
|----|--------|---------|
| 메인 | 집 | 기본 선택 |
| 신규 입사자 | 사람+ | 주황색 강조 |
| 가전 안전 | TV | |
| 에어컨 안전 | 바람 | |
| 창고 안전 | 창고 | |
| 안전 규정 | 클립보드 | |
| 중대재해알림 | 경고 | |
| 안전미준수사례 | 말풍선 | |
| 관리자메뉴 | 사람+톱니 | |

**왼쪽 사이드패널**:
- "메인" 레이블
- 녹색 "+ 게시물 작성" 버튼
- 아코디언 게시물 목록 (탭별 다른 목록)

**오른쪽 콘텐츠 뷰어**:
- 초기: 말풍선 아이콘 + "게시물을 선택해주세요."
- 선택 후: 제목, 작성자, 날짜, 조회수, 본문 내용 표시
- 수정/삭제 버튼, 유튜브 영상 임베드

**게시물 작성 모달**: 제목, 카테고리, 내용, 영상URL 입력

아이디/비밀번호 아무 값이나 입력 → 대시보드 이동  
`admin` 입력 시 '관리자'로 표시
